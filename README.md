# Mecanizou Protótipo Runner v4

Protótipo React/Vite para testar o fluxo de Compras, Logística 1ª Perna, Expedição e Última Milha.

## Rodar localmente

```bash
npm install
npm run dev
```

Abra a URL mostrada no terminal, geralmente `http://localhost:5173`.

## Observação

A v4 usa a chave `mecanizou-prototipo-v4` no `localStorage`. Use o botão de reset no topo para voltar aos dados iniciais.
